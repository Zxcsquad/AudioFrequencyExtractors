О программе

Audio Frequency Extractor — это мощный инструмент для точного извлечения и обработки частотных полос из аудиофайлов. Идеален для работы с сабвуфером, баксом и других частотных диапазонов.

Программа использует передовые алгоритмы фильтрации (фильтр Баттерворта 4-го порядка) с двусторонней обработкой для минимизации артефактов и искажений фазы.

❓ Часто задаваемые вопросы

Какие форматы файлов поддерживаются?

На входе: WAV, FLAC, OGG, MP3. На выходе: WAV, FLAC. Программа автоматически сохраняет исходный формат.

Какая минимальная частота дискретизации?

Рекомендуется 44100 Hz и выше. Для работы с частотами выше 10000 Hz используйте 48000 Hz или больше.

Можно ли использовать в коммерческих целях?

Бесплатная версия — для личного использования. Pro версия включает коммерческую лицензию.

Как установить зависимости?

Выполните: pip install -r requirements.txt

Есть ли версия для командной строки?

Да! Используйте python process_subwoofer.py для интерактивного режима.
