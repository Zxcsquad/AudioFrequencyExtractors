## 🚀 Быстрый старт

1. **Откройте в браузере:**

   - Или откройте через браузер: `File → Open File → index.html`

2. **Локальный сервер (опционально):**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```
   Затем откройте: `http://localhost:8000`
